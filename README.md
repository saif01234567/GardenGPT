# 🌱 GardenGPT

AI-Powered Smart Gardening Assistant built using Python, Streamlit and PlantNet API.

## Features

* Upload plant images
* Identify plant species
* View scientific and common names
* Confidence scoring
* Plant care recommendations
* Water requirements
* Sunlight requirements
* Soil recommendations
* Fertilizer suggestions
* Fun facts

## Technologies Used

* Python
* Streamlit
* PlantNet API
* Git
* GitHub
